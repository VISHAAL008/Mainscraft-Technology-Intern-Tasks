# House Price Predictor

A machine learning project developed as part of the **Maincrafts Technology AI & Machine Learning Internship**.

## Overview

This project predicts California house prices using a **Linear Regression** model trained on the California Housing Dataset.

## Features

* Data preprocessing
* Exploratory Data Analysis (EDA)
* Linear Regression model
* Model evaluation (MAE, RMSE, R² Score)
* Trained model saved as `.pkl`

## Tech Stack

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Seaborn

## Results

* **MAE:** 0.533
* **RMSE:** 0.746
* **R² Score:** 0.576

* `train.py` / `app.py`
